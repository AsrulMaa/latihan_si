Aplikasi yang di butuhkan
1. Webserver & database server / bisa menggunakan xampp, wamp
	Link Download Xampp : https://www.apachefriends.org/download.html
2. Text Editor / IDE, 
	Text Editor : Notepad++, Sublime Text
	IDE	: Netbeans, Atom, Dreamweaver
	Link Download Sublime text : https://www.sublimetext.com/3

1. copy folder project, dan masukkan ke dalam folder htdocs ( C:\xampp\htdocs )
2. Hidupkan Webserver dan database server melalui xampp control panel, 
	tekan tombol start pada apache dan mysql.
3. Buka web browser, lalu ketik pada address bar, localhost/phpmyadmin
4. selanjutnya buat database baru, klik tombol database, lalu masukkan nama database 
	"latihan_si" tanpa tanda kutip, selanjutnya klik tombol import, lalu pilih file
	"latihan_si" yang ada pada folder latihan_si, jika sudah, tekan tombol go 
5. Untuk mengakses hasil latihan, Buka Browser, lalu ketikan : localhost/nama_folder_yang_tadi_di_pindahkan_ke_htdocs
